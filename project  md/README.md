# Ordner `project md` – alle Doku & Notizen

Hier liegt die **Projektdokumentation** gebündelt (Markdown + Text).  
Code bleibt unter **`src/`** – dieser Ordner ist nur für **Lesen / Planen**.

**Neu / zentral:** **[MENTOR-REGELN.md](MENTOR-REGELN.md)** – wie die KI dich coacht (Fullstack, Fundamente, kein blindes Copy-Paste, Review-Stil, Hausaufgaben).
